# 🌤️ Weather Dashboard

A modern **Weather Dashboard** built using **React (Vite)** that fetches real-time weather data from the **OpenWeatherMap API**.  

Users can search for Indian cities and view live weather details including temperature, humidity, pressure, and weather conditions.

---

## 🚀 Features

- 🌡️ Real-time temperature (°C)
- 💧 Humidity percentage
- 📊 Atmospheric pressure
- ☁️ Weather condition with description
- 🔍 City-based search (India focused)
- ⌨️ Search on Enter key
- 🔐 Secure API key handling using `.env`
- ⚡ Fast performance with Vite

---

## 🛠️ Tech Stack

- **Frontend:** React (Vite)  
- **HTTP Client:** Axios  
- **API:** OpenWeatherMap REST API  
- **Styling:** CSS  
- **Version Control:** Git & GitHub
